Fundamentals of Computer Engineering

Introduction

The course "Fundamentals of Computer Engineering" covers the fundamental concepts of computer engineering. It will focus on computer architecture, low-level programming, and the interaction between hardware and software.

Course Objectives

  • Understand how computers work at the hardware level.
  • Develop skills in low-level programming languages.
  • Learn about the main computer architectures.

Main Content

  • Basic computer architecture.
  • Low-level programming languages (assembly language).
  • Data structures: stacks, queues, and lists.

Learning Outcomes

By the end of the course, students will understand the fundamental principles of computer system design and be able to write simple programs in low-level languages.
