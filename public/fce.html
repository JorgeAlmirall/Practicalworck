Fundamentals of Computer Engineering

Introduction

The "Fundamentals of Computer Engineering" course provides a comprehensive overview of key computer engineering concepts. It focuses on the relationship between hardware and software, offering foundational knowledge for future study in the field.

Course Objectives

  • Understand the evolution and current landscape of computer science and engineering.
  • Learn about computer engineering as both a field of study and a profession.
  • Gain an understanding of hardware components, software systems, and their interactions.
  • Explore modern trends, including big data, operating systems, and software development.

Course Modules

  • Module 1: Background and Historical Context – The development of computer science and its relevance today.
  • Module 2: Hardware and Software – Core components of computer systems, including their functions and interdependencies.
  • Module 3: Information and Big Data – The role of information and data management in modern engineering.
  • Module 4: Introduction to Software Development – Basic principles of creating software applications.
  • Module 5: Web Programming and HTML – Overview of web development and structure using HTML.
  • Module 6: Introduction to CSS – The basics of styling web pages using CSS.

Learning Outcomes

By the end of the course, students will have a strong foundation in both hardware and software systems, be familiar with web development basics, and understand how data is managed and utilized in computer engineering.
